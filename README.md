# 🤖 AgeMysticBot

AgeMysticBot is a Telegram bot that analyzes user age and generates fun insights like estimated birth year and personality traits.

## 🚀 Features

* 🎂 Age analysis
* 📅 Birth year estimation
* 🧠 Personality insights
* 📱 Interactive Telegram buttons

## ⚙️ Tech Stack

* Python
* python-telegram-bot
* HTTPX

## 📦 Deployment

This bot is designed to be deployed on Railway.

## 🔐 Environment Variables

Create a variable:

BOT_TOKEN=your_telegram_bot_token

## ▶️ Run Locally

pip install -r requirements.txt
python bot.py

## 💡 Note

Do not share your BOT_TOKEN publicly.
